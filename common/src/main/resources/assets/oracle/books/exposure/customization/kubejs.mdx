---
title: KubeJS Integration
---

Exposure exposes (heh) three events to KubeJS:

`ExposureEvents.addEntityInFrameExtraData`- can be used to add additional properties to frame's entity in frame extra data\
`ExposureEvents.modifyFrameExtraData` - can be used to add additional properties to frame's extra data\
`ExposureEvents.frameAdded` - somewhat similar to [FrameExposedTrigger](triggers/frame_exposed_trigger)
