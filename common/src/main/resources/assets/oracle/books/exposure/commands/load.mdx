---
title: Load Command
---

`/exposure load <capture_properties>` - loads an image from a file or URL.\
*projection* property of [Capture Properties](../customization/components/capture_properties) must be specified.

#### Example:
`/exposure load {projection:{path:"D:/images/cat.png"}}`
