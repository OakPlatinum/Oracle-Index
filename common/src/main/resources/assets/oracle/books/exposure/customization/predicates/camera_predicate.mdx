| Property | Type | Description |
|-|-|-|
| camera | ItemPredicate | Checks camera item stack |
| film | ItemPredicate | Checks Film attachment of the camera |
| flash | ItemPredicate | Checks Flash attachment of the camera |
| lens | ItemPredicate | Checks Lens attachment of the camera |
| filter | ItemPredicate | Checks Filter attachment of the camera |
| location | LocationPredicate | Checks current position of the camera |
