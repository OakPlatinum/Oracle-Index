### Exports exposures as PNG files to `<instance>/exposures` folder on client's PC.

<div name='NOTE' style={{ display: 'inline-block', margin: '2px' }}>
    <div style={{ display: 'flex', alignItems: 'top', backgroundColor: '#232425', borderRadius: '6px', padding: '8px 12px' }}>
      <RocketIcon style={{ color: '#F4EFE9', marginRight: '8px' }} width={20} />
      <span style={{ color: '#F4EFE9', fontSize: '15px' }}>Exported files will be organized in folder with world name, unless turned off in a config.</span>
    </div>
</div><br/>

`/exposure export id <id> <size> <look>` - exports single exposure.\
`/exposure export all <size> <look>` - exports all exposures of the current world. Asks for confirmation.\
`/exposure export stop` - stops export that is currently in progress.

`<size>` - size modifier of exported image. Exporting 320px (default) exposure with size x2 will produce 640px png.\
`<look>` - style of the exposure. Exported file will be suffixed with it so you can export multiple looks without them overriding each other.
