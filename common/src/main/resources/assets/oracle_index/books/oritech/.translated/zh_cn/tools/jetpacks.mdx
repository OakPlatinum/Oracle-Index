icon: oritech:basic_jetpack

<ModAsset location="oritech:basic_jetpack" /> 是时候飞向天空了。Oritech 提供了不同类型的喷气背包。所有喷气背包都可以使用能量或涡轮燃料飞行。要为其充能，可以使用任何能量存储装置或充电器。要填充内部涡轮燃料箱，则需要使用充电方块。

当使用涡轮燃料时，喷气背包的飞行速度会快得多。外骨骼喷气背包比基础版本更快，上升速度更快，并且在不向上飞的情况下也能增加前进速度。

喷气背包无法手动开关。当按住跳跃键时，喷气背包会自动激活。

基础喷气背包和外骨骼喷气背包都有鞘翅版本。在这种情况下，喷气背包仅在用户使用鞘翅飞行时激活，并提供向上的推力，而不像使用烟花那样。
