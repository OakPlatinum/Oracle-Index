## 概述

**Oritech反应堆** 是一种生成大量能量的绝佳方式。它们消耗铀或钚燃料棒，并根据相邻的反应堆燃料棒和中子反射器的数量产生RF（射频）能量。
然而，保持其充分冷却非常重要。

燃烧燃料会生成热量，有多个组件用于管理反应堆的热量：
- **热管**：从燃料棒中吸收热量并沿管道传输。
- **反应堆散热口**：从最热的相邻组件（例如燃料棒或热管）中吸收部分热量，并将其排出反应堆。
- **吸热装置**：从所有相邻组件中吸收固定数量的热量，但需要持续供应冷却剂物品才能正常工作。

反应堆允许一定程度的热量积聚。理想情况下，适当冷却的反应堆不会产生任何泄漏热量。然而，某些组件（如散热口）在高温环境下效率更高。
如果产生的热量过多且无法移除，燃料棒可能会过热，导致核熔毁。如果燃料棒温度超过2000°C并持续一段时间，将会触发熔毁。
在达到这些温度之前，警告警报会启动。根据反应堆的大小和燃料棒的数量，熔毁会导致不同程度的爆炸。

<Callout variant="info">
    对于想要安全探索新反应堆设计和变体的Oritech科技股份有限公司的新手科学家来说，有一个名为“安全反应堆”的配置选项可用。
    当反应堆过热时，该选项会使反应堆进入冷却期而不是爆炸。
</Callout>

反应堆需要由矩形墙壁构成，边缘由**反应堆墙**方块组成。反应堆的最大尺寸为每个方向**64**个方块。
需要在墙壁上放置一个**反应堆控制器**。**能量端口**也可以放置在侧墙上。
所有其他非端口组件必须放置在反应堆内部。反应堆可以具有任意高度，但内部结构在垂直轴上必须相同。
这意味着您可以**在二维空间中设计反应堆，然后根据需要垂直扩展**。任何内部高度为1的反应堆也可以类似地工作于内部高度为10的情况。
然而，能量生产、冷却剂和燃料消耗将相应地按比例缩放。
