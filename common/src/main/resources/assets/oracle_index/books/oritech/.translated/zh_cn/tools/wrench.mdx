### 描述
扳手是一种用于编辑和破坏管道的工具。它用于更改管道每个方向上的连接，以及用于破坏管道。

### 使用方法

#### 切换管道连接
所有管道在各自的方向上都可以断开或连接。要切换管道连接：
- 您必须右键点击现有的连接以禁用它。
- 要启用连接，您必须右键点击管道的中心，并点击您希望启用的方向对应的侧面。

#### 破坏管道
要破坏管道，您必须按住Shift并右键点击管道（同时手持扳手）。管道会立即掉落为物品。
或者，您可以像使用镐一样用扳手破坏管道。扳手在破坏管道时具有更高的挖掘速度。
