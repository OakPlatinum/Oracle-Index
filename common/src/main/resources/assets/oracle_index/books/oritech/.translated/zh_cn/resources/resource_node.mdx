资源节点分布在世界的各个地方，出现在基岩层上方不远处，呈簇状分布。
它们包含所有类型的主世界矿石，通常被一小片[原油](../resources/crude_oil)覆盖。

资源节点只能使用[基岩层钻机](../interaction/deepdrill)进行挖掘。

在地表，[巨大矿团](../resources/ore_boulder)的出现标志着下方可能存在资源节点。
巨大矿团还指示了该节点簇中可以找到的矿石类型。
