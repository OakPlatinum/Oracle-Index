## 描述
无人机港允许您使用飞行无人机远距离运输物品和流体。它要求在起飞和降落位置都建立并供电无人机港。

## 使用方法
### 设置目标无人机港
建成后，使用目标指示器物品指定目标港。通过按住Shift键并右键点击目标港进行绑定。然后，在您希望发送材料的港上打开UI，并将指示器放入特殊物品槽中。

### 距离限制
目标港必须至少相距50个方块，并且其所在区域需要保持区块加载状态。

### 发送/接收材料
一个无人机港只能向一个特定的目标港发送材料，但一个港可以从多个港接收材料。然而，每个无人机着陆需要几秒钟时间，因此如果材料到达得太频繁，一个接收港可能会因多个港同时发送而过载。

### 行程成本
无人机送达材料所需的时间是固定的，与飞行距离无关。然而，能量成本随距离增加。能量成本计算公式如下：

`sqrt(距离) * 50 + 1024`

### 发送流体
要发送流体，必须为无人机港添加流体附加组件。发送和接收港都必须有流体附加组件，无人机才能运输流体。

### 红石控制
可以添加 **红石控制单元插件** 以允许无人机港通过红石信号进行控制，并通过红石控制单元插件发送比较器信号。

**模式：**
- **能量测量模式 (Power Measure):** 比较器信号强度基于无人机港中存储的能量量。
- **物品栏槽位测量模式 (Inventory Slot Measure):** 比较器信号强度基于目标槽中最大数量的某种材料，无论是物品还是流体罐中的流体。
- **进度测量模式 (Progress Measure):** 比较器信号强度基于无人机港的进度。
- **活动测量模式 (Activity Measure):** 比较器信号强度基于无人机港是否处于空闲、发送或接收状态。
- **输入控制模式 (Input Control):** 向红石控制单元插件输入红石信号以禁用无人机港发送或接收材料。
