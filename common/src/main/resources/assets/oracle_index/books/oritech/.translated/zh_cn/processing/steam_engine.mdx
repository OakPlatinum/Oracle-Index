除基础发电机外外的所有发电机都能安装蒸汽锅炉插件以产出蒸汽。在安装蒸汽锅炉插件后，发电机不会直接产出能量，而是会在向其输入水的情况下以两倍的速度产出蒸汽。水/蒸汽可以分别通过流体管道直接从蒸汽锅炉插件而不是发电机输入/输出。

想要利用蒸汽发电，您首先需要建造一个蒸汽引擎。它接收蒸汽并输出（冷凝）水，但是输出的冷凝水量相较于用于产出蒸汽的水量会有大约20%的损耗，所以您仍然需要一个稳定的水源。

<Callout variant="info">
    蒸汽机将会在它们的水或能量存储完全装满时停止工作。始终确保水可以被排出。
    用一些储罐，比较器以及红石，你或许能建造一个自生循环系统。
</Callout>

多个蒸汽引擎可以组成蒸汽引擎阵列，其共享阵列中第一台蒸汽引擎的能量、蒸汽和冷凝水储量。它们会协同工作。蒸汽引擎的摇臂转速取决于实际蒸汽储量，更多的蒸汽会提供更大的压力进而使蒸汽引擎能够更快地运作。其运行速度与实际蒸汽储量相较于最大蒸汽储量的比值呈线性关系，当比值为1时，即实际蒸汽储量等于最大蒸汽储量时，蒸汽引擎运行速率为1000%。

然而，引擎能效基于运行速度。能效越高，单个蒸汽引擎能够产出的能量就越多，能效在运行速率为700%左右时达到顶峰。更高或更低的运行速率都会降低能效。能量将会从红色接口输出，而流体接口为蓝色。
