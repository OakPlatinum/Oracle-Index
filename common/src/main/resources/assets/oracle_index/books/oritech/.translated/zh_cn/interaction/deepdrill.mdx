基岩层钻机可以在[资源节点](../resources/resource_node)被发现的位置开采组员。
该[多方块结构](../processing/multiblocks)机器只能在资源节点上方运行，且必须使用[末影激光器](../interaction/enderic_laser)来为其供能。

当右键时，将显示下方被发现的资源节点。它会显示下方3x3区域的第一个可见的资源节点，且能最多达3格深。

资源节点会提供无尽的资源，且不能被破坏。

基岩层钻机工作时，其会开采与下方3x3范围内的资源点类型相同的矿物，每次的矿物产出类型随机。不要求范围内所有方块均为资源点。

试着找到一个能覆盖最多节点的位置以求最高产量。
