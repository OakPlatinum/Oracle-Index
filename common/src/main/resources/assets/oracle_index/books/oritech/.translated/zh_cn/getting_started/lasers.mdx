title: 激！光！以及辐水晶

在你更进一步之前，你会需要一些[辐水晶](../resources/fluxite)。要想获得它，用[末影激光](../interaction/enderic_laser)使紫水晶簇过载。

制造一个末影激光器将会基于之前的所有步骤。它需要一系列的合金，装配的组件，甚至一些如[末影化合物](../resources/enderic_compound)的物品以及用[离心机](../processing/centrifuge)和[粉碎机](../processing/pulverizer)制造的碳纤维。

用你新到手的末影激光器，[目标标识器](../tools/target_designator)和[发电机](../processing/generators)，你就可以开始制造一些辐水晶了。

你可以用末影激光器挖掘紫水晶簇来获得辐水晶。使用目标标识器定位一个紫水晶簇。紫水晶母岩将不会被破坏，而是会被充能，让紫水晶簇长得更快。
